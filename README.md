# Discord News Bot

Discord бот для публикации новостей на сервере с картинками, ссылками и цветами.

---

## 🚀 Запуск

### Требования
- Python 3.9+
- disnake
- Discord токен бота

### Как запустить

**Windows (.bat файлы):**
```bash
start.bat          # обычный запуск
startTest.bat      # режим отладки
```

**Или через консоль:**
```bash
py bot.py          # обычный запуск
py bot.py --test   # режим отладки
```

## ⚙️ Настройка

**Создай файл `.env`:**
```
token=твой_токен_бота
```

**Файл `cfg.json` уже есть в проекте**

## 📋 Команды

### `/auth_news` — Первичная настройка

Регистрирует сервер и добавляет первую роль редакторов.

```
/auth_news роль:@Редакторы
```

**Доступ:** Владелец, Администраторы

---

### `/send_news` — Опубликовать новость

Публикует новость в канал.

```
/send_news канал:#новости роль:@Важное картинка:[файл] цвет:#FF0000
```

**Параметры:**
- `канал` — куда отправить (обязательно)
- `роль` — кого упомянуть
- `картинка` — файл изображения
- `ссылка` — URL картинки
- `цвет` — HEX код (#FF5733)

После команды откроется форма для заголовка и текста.

**Доступ:** Владелец, Администраторы, Редакторы

---

### `/preview_news` — Предпросмотр

Показывает как будет выглядеть новость (только вам).

```
/preview_news ссылка:https://example.com/pic.jpg цвет:#00FF00
```

**Доступ:** Владелец, Администраторы, Редакторы

---

### `/cfg_addrole` — Добавить роль

Даёт роли доступ к публикации новостей.

```
/cfg_addrole роль:@Модераторы
```

**Доступ:** Владелец, Администраторы, Редакторы

---

### `/cfg_dellrole` — Удалить роль

Убирает доступ у роли.

```
/cfg_dellrole роль:@Модераторы
```

**Доступ:** Владелец, Администраторы, Редакторы

---

### `/cfg_info` — Информация

Показывает текущие настройки.

```
/cfg_info
```

**Доступ:** Владелец, Администраторы, Редакторы

---

### `/cfg_reset` — Сброс настроек

Полностью сбрасывает конфиг (с подтверждением).

```
/cfg_reset
```

**Доступ:** Владелец, Администраторы

---

## ⚠️ Ошибки

| Сообщение | Что делать |
|-----------|------------|
| Сервер не зарегистрирован | Выполни `/auth_news` |
| У вас нет прав | Попроси права у администратора |
| Роль уже есть | Используй другую роль |
| Неверный HEX формат | Пиши цвет как `#FF5733` |
